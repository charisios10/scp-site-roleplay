SCP: Site Roleplay

This is a small page for new players of the SCP: Site Roleplay to learn more things about the game.

1.1 Alarms

Alarm Types

Alarm Type Description
Breach Alarm When a breach/es is detected, activate this alarm. (Containment 1 doesn't count)
CD Escape When 3 or more CDs escape the CDC hub.
CI Entered Site When a CI member has entered the site or has been spoted somewhere in the site.
Lockdown When a lot of alarms are detected at the same time, or the SID+ has lost control of the site.

Broadcast Examples - Templates

Broadcast Type Used in Message
All clear When the site status is clear. /broadcast All operations may continue as normal / {Testing Status: Allowed} 035 And con x need sid+ perm/ SCP: Foundation
Breach type 1 When a breach/es is detected with the danger of medium type. /broadcast All operations may continue with caution / {Testing Status: May continue with caution} All test's are prohibited from con // and upper./ SCP: Foundation
Breach type 2 When there is breaches with a very high danger. /broadcast All site operations are prohibited / {Testing Status: Not Allowed} / SCP: Foundation
Lockdown When the site is in lockdown. /broadcast All site operations are prohibited / {Testing Status: Not Allowed} / SCP: Foundation
CD Escape When 3 or more CDs escape the CDC hub. /broadcast All site operations are prohibited / {Testing Status: Not Allowed} / SCP: Foundation
CI Entered Site When a CI member has entered the site or has been spoted somewhere in the site. /broadcast All site operations are prohibited / {Testing Status: Not Allowed} / SCP: Foundation
Meeting Call When a meeting is called. /broadcast Please all sid+ members go to the meeting room / {Testing Status: Not Allowed} / SCP: Foundation
Nuke Detonation When the nuke is detonated. /broadcast Nuke detonated / {Testing Status: Not Allowed} / SCP: Foundation

What to do on Nuke Detonation

In case of the site being rioted with breaches and it starts to get out of control the council or the sids and o5s can iniciaty the warheads protocol. The warhead protocol is a nuke that resets all scps and recontains them, this protocol can be activated when 3+ breaches are out for a lot of time. Also the protocol can be activated with 2 o5s as they can only take the keys of anyone in the site. To survive the nuke you should head to the nuke shelter wich is located in the con 2 entrance at your left as you enter from the con 1. The nuke as also stated in the rules must be used as a last resort and in case of abuse there will be heavy consequences.
